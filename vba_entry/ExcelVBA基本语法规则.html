# 几个术语

# 变量

变量来源于数学，是计算机语言中能储存计算结果或能表示值抽象概念。变量可以通过变量名访问。在指令式语言中，变量通常是可变的；但在纯函数式语言（如Haskell）中，变量可能是不可变（immutable）的。——百度百科

上面的定义比较生涩拗口，我觉得 方骥老师 的比喻更加生动贴切。

这里的变量就像方程式里的 x, y，一开始你列了方程，但是暂时还没法算出一个数，之后只要把x变成具体的数字就可以算出一个值了。

# 过程

网上关于过程的定义：过程组被作为一个整体，指示Excel中如何执行特定任务执行的语句。执行的任务可以非常简单或非常复杂的，这是一个很好的做法，把程序分成较小的部分。

# 模块

网上对模块的定义：模块是其中代码被写入的区域。

VBA编辑窗口中的左侧有一个工程栏，里面显示的都是工程项目，在列表中是一个个模块。

# 变量

# 为什么要使用变量

运用变量来编写带有变动参数的语句代码，在为这些变量赋值之后就可以具体执行这些语句。

# 常用变量类型

  • Variant（万用型）
  • Date（日期）
  • Boolean（布尔类型/逻辑值）
  • Object（对象）
  • String（字符串）
  • Integer（整数）
  • Long（长整数）
  • Single（单精度浮点数）

Excel中的Integer的范围是从负的30,000+到正的30,000+，并不包含所有的整数。因此如果要涉及到绝对值更大的数值，就要使用Long类型

# 申明变量

在经过之前的动手，我发现其实在VBA中不用申明也可以成功的运行，不像Java等语言，如果不申明变量就根本无法运行。但我们需要知道，虽然不申明变量的类型也可以正常的使用，但是在使用变量之前申明变量会使程序变得更严谨。 ExcelVBA中申明变量要用Dim语句 例子：

Dim r As Integer
Dim newbook As Workbook

有的变量可以用符号来代替，如Integer可以用%来代替。

可以用符号来代替的变量 符号
字符串String $
整数Integer %
长整数Long &
单精度浮点数Single !

例如：

Dim r%

申明变量后应当注意的问题：

  • 所有变量赋值应当和变量的类型相匹配
  • 对大部分的变量而言，赋值的时候只用=就可以了，但是对象类型的变量在进行赋值的时候必须加上Set才能赋值。

# 不同级别的变量

# 过程

最常见的在子程序的编写过程中申明的变量就是过程级别的变量。过程级别的变量只有在当前自己编写的这个Sub中会使用，而不会和其他的Sub共享。

# 模块

在模块中申明的变量可以在不同的过程之间传递。简单的来说就是，在一个编辑窗口的最顶部（所有的Sub之上）申明的变量就是模块级别的变量。

# 公共

模块列表中有不同的文件夹，其中在Excel对象文件夹下的模块和在模块文件夹下的模块是有区别的，后者是全局类型的模块，在后者编辑窗口中可以申明公共变量。公共变量是全局变量，可以在不同的模块之间传递。

# 过程

ExcelVBA的过程有三种：

  1. 函数Function
  2. 子程序Sub
  3. 属性Property

这三种过程里最后一种不太常见，前面两种有一个明显的不同之处：

  • Function函数除了执行特定的语句外还会返回一个特定的结果
  • Sub则只是执行语句，不会返回一个结果

# 分类

  • 逻辑运算符
  • 算数运算符

# 逻辑运算符

符号 意义
And 或计算
Or 与计算
Not 非计算

# 算数运算符

符号 意义
+ 加
- 减
* 乘
/ 除
^ 幂指运算
\ 整除运算*
Mod 取余数运算*

最后面两个运算可能没怎么听过，举个例子： 5\2的结果是2 5 Mod 2的结果是1

对不同的运算符的优先级别无法确定的话就可以使用括号（和数学中的用法很像，括号内的优先运算）

# 连接运算符

连接运算符比较少，只有两个：

  • +
  • &

其中+不仅能表示算数运算中的加法，也能用来连接不同的字符串。&之前也用过，比起+来我用&更多一些
