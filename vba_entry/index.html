# 自己的经历分享

在正式开始分享VBA相关知识之前，我希望简单分享一下我的个人经历，因为VBA再怎么说也是一种编程语言，而很多人对编程有着天生的畏惧感，觉得很难，觉得自己不行，在这里希望我的经历能给你一些鼓励。我本科是会计专业，学了两年才发现自己真的不想做个会计，于是就天天想着怎么转行，我对Excel的高效简洁非常有兴趣，就报了一门学习用Excel做财务报表的课，学了觉得不过瘾，又自己上网去钻研各种Excel的方程，每次发现新的用法，用组合的公式解决一些别人非常头疼的问题，就心里暗暗窃喜。但是有很多结构相似又重复的表格总是要我人工修改和键入公式也太麻烦了，逐渐的我越来越来，最后听说了VBA，第一次接触 循环 的时候就被它的高效给惊艳到了，原本一个简单的方程因为有了完整编程语言的存在而变得更加强大，迸发出无限的力量和可能。我慢慢地就迷上了编程，从原本一个没有任何编程基础的人，一路学完VBA又学Python又学JS，从没想过一次偶然的机会就给了我转行的可能。其实有时候真的不用吓自己，也不用焦虑什么时候才能学会，就一步一步地去学就好，也许未来回过头来就发现，原来自己已经走出去那么远了。
