# 数组

把单个的数字组合在一起，分为静态数组和动态数组两种。

# 静态数组

一般在事先知道数组中有多少个数字的情况下会选择使用静态数组。

# 动态数组

一般在事先不知道数组中会有多少个数字的情况下回选择使用动态数组。

# 声明

# 静态数组

对静态数组的声明方式有两种：

  1. Dim Arr(10) As Integer 括号中的10代表最大下标是10，即数组中最大的数字编号为10。VBA中的数组默认从0开始编号，所以上面这个数组中的数组编号从0~10，总共有11个数字。
  2. Dim Arr(1 to 10) As Integer 这种申明方式同时定义了最小编号和最大编号，这个数组中只有10个数字。

# 动态数组

对动态数组的声明也有两种方式：

  1. Dim Arr()这种方式申明的数组是范写，适用范围更广
  2. Dim Arr(1 to S)其中的S是一个变量

# 赋值

# 静态数组

# 直接赋值

Dim arr1() as Interge
arr1(1)=5
arr1(2)=3
arr1(3)=12

# 通过循环结构给静态数组赋值

Dim arr1(1 To 3) As Integer
Dim i As integer
For i = 1 To 3
	arr1(i) = Range("A" & i)

Next i

# 通过循环结构给动态数组赋值

Dim arr2() As Integer
Dim i As integer
For i = 1 To 3
	ReDim Preserve arr2(1 to i)
	arr1(i) = Range("A" & i)

Next i

# 动态数组

# 通过函数赋值

Dim arr2()

arr2 = WorksheetFunction.Transpose(Range("A1:A3").Value)

# 利用Array函数赋值

Dim arr2()
arr2 = Array(5,3,12)
