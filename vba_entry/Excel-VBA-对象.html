# 对象

# 是什么

所谓对象就是我们要对什么东西进行操作。例如，我们要对单元格进行填充上色，又例如我们复制当前一个工作表等等。 之前完成的VBA里我们仅仅在VBA的编辑窗口里改动了第一行和最后一行，给整个代码块套上了一个循环结构，但是中间的代码究竟代表什么我们没有去深究。 当我们理解了对象之后，我们就可以读懂中间的代码究竟对哪些东西进行了操作和修改。

# 具体分类

主要有以下几种类型：

  • Application
  • Workbooks
  • Sheets
  • Worksheets
  • Charts
  • Range

# Application

Application指的是整个运行程序，当我们打开一个或者几个Excel表格文件的时候，这里的application指的就是Excel。

# Workbooks

WorkBooks指的是我们在Excel表格中创建的工作表的合集。

# Sheets

Sheets这个单词可能比起前几个会看起来比较眼熟，因为我们每次新建一个Excel的时候，左下角的页面标签就是Sheets1。Sheets就是工作表，与前面的工作簿有一字之差。就跟日常生活中一样，“表”是一张一张的，当很多张“表”被钉在一块，就成了一本“簿”，我们也可以这样来理解 Sheets 和 WorkBooks 的区别。

# Worksheets

这里又出现了一个Worksheets，字面意思是工作表。那是不是我们选中的表就叫工作表呢？并不是的，其实Excel中有很多种的表格类型，当我们右键单击标签，选择插入时，会出先很多种表格类型让我们选择。 在图中我们可以看到除了“工作表”以外，还有“图表”和“宏表”等很多类型的表。我们平常用的最多的就是“工作表”，这里的Worksheets也特指“工作表”这一类型的表格。而上面提到的Sheets则泛指所有类型的表。

# Charts

原理与上面的Worksheets差不多，Charts就是特指“图表”类型的表。

# Range

Range就是单元格的意思，比起上面的几个词来说应该是比较好理解的了。
