与之前的二维表转换成一维表相似，这次将一对多的数据转换成一位表格也是遍历数据然后添加在新的工作表中。

首先，我们要有一个概念，一对多数据表到底长什么样？ 接着我们讲一下思路，这一次与二维表的区别主要在于在号码和城市后面的套餐不是仅仅只有一个，而是一个号码对应着多个套餐，有的号码对应的套餐没有那么多，它后面的单元格就有空缺。所以我们选择遍历所有的套餐单元格，对单元格的内容进行判断，如果是空单元格，那么该号码没有套餐2或者套餐3，就不用管它，如果该单元格有数据，那就选择在新的工作表中加入一行，并在套餐写入该套餐对应的号码和城市，组成一维数据表。 先贴上这次的代码：

Sub 一对多转一维()

arr = ActiveSheet.UsedRange
rw = UBound(arr, 1)
col = UBound(arr, 2)

ReDim brr(1 To rw * (col - 2), 1 To 3)
n = 2
For i = 2 To rw
    For j = 3 To col
        If Cells(i, j) <> "" Then
            brr(n, 1) = arr(i, 1)
            brr(n, 2) = arr(i, 2)
            brr(n, 3) = arr(i, j)
            n = n + 1
        End If
    Next j
Next i

brr(1, 1) = "号码"
brr(1, 2) = "城市"
brr(1, 3) = "套餐"

Sheets.Add.Name = "转换结果"
Sheets("转换结果").Range(Cells(1, 1), Cells(n - 1, 3)) = brr

End Sub

讲一讲其中和上一次二维表转换相比不同的地方，其中主要加入了一次判断和UBound。 UBound指的是选取数组的上标，数组中最小值是数组的下标，上标则恰好相反。
