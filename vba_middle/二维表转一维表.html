今天主要学习了如何通过VBA将二维表转换成为一维表。先不急着讲学到了什么方法和代码，我觉得关于二维表和一维表的定义就可以先来讲一讲。 可以看出二维表与一维表有很大的不同，二维表中每个数据都对应着两个字段名称，而一维表中值对应自己这一行的数据。简单说，看二维表总要纵看一次横看一次，而一维表只用横着看一次就行。

在《三张表格走天下》中我读到过小蚊子老师这样的总结，做数据分析需要三张表，一张是参数表，一张是录入表，最后一张是透视表。其中包括最多的原始数据的录入表就要采用一维表格的形式。为什么呢？因为一维表格更加的简单，更加适合采用筛选、排序等方法对数据进行提取和加工。

一维数据表在数据分析的工作中有重要的作用，但是普通人似乎更习惯二维表，因为生活中见得最多的还是二维表。如何进行转换呢？我们的思路是： 其中字段一和字段二都是标题名称，所以字段一我们固定列，按行递增来取用并赋值给一维表，对于字段二我们选择固定行，按列递增来提取并赋值。中间的数据字段三则选择利用双层循环的遍历方式来进行提取和赋值。

我们使用以下代码来实现我们的思路：

Sub 二维表转一维表()

Dim arr
Dim rw
Dim col
Dim brr
Dim n
Dim i
Dim j

arr = ActiveSheet.UsedRange
rw = ActiveSheet.UsedRange.Rows.Count
col = ActiveSheet.UsedRange.Columns.Count

'转换过程
ReDim brr(1 To rw * col, 1 To 3)
n = 2
For i = 2 To rw
    For j = 2 To col
        brr(n, 1) = arr(i, 1)
        brr(n, 2) = arr(1, j)
        brr(n, 3) = arr(i, j)
        n = n + 1
    Next j
Next i

'补齐字段名
brr(1, 1) = "地区"
brr(1, 2) = "操作系统"
brr(1, 3) = "使用人数"

'新建表格储存转换结果
Sheets.Add.Name = "转换结果"
Sheets("转换结果").Range(Cells(1, 1), Cells(n - 1, 3)) = brr


End Sub

可能有人不明白为什么不直接创建一个新表提取赋值给新表的单元格，却要选择先赋值个一个中间数列brr。我想，这个原因大概是因为对单元格直接进行操作比较慢，并且写出的代码也会相对更加繁琐，所以先创建一个新的数列，最后一次性赋值，这样程序的速度会更快。（在处理的表格数据量和数量越来越大的时候这种优势就会越来越明显）
